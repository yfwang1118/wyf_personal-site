# Wang Yufeng Personal Site

Bilingual personal website for Wang Yufeng, built with Next.js and Tailwind CSS.

## Local Development

```bash
npm install
npm run dev
```

The site uses mirrored English and Chinese routes:

- `/en`
- `/zh`
- `/en/work`
- `/zh/work`
- `/en/writing`
- `/zh/writing`
- `/en/about`
- `/zh/about`
- `/en/contact`
- `/zh/contact`

## Deploy

This project can be deployed directly on Vercel as a standard Next.js app.
